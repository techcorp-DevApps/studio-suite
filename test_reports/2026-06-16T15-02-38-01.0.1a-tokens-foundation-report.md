## Decisions & notes

- **pnpm version pinned to `10.34.3`, not latest.** Latest stable is `11.7.0`, but
  pnpm 11 requires Node ≥ 22.13 (it imports `node:sqlite`); this environment / CI
  runs Node 20. pnpm 10.34.3 supports Node ≥ 18.12. `packageManager` +
  `pnpm/action-setup` keep CI and local identical.
- **`.npmrc` reconciliation.** `web/.npmrc` held only `legacy-peer-deps=true`, an
  npm-only flag. pnpm defaults (`auto-install-peers=true`, non-strict peers) cover
  the React-19 peer case, so the flag was dropped and **no root `.npmrc` was
  created** — no hoisting/node-linker settings were needed by any build, so none
  were added speculatively.
- **`git fetch` could not reach origin** in this environment (SSH key auth blocked);
  the cached `origin/main` is exactly the pinned `42b1a0b`, and this task is local +
  owner-gated (no push/PR/merge), so this did not affect the work.
